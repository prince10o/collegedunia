# College Dunia Project

Welcome to the College Dunia project! This project is a web application that provides comprehensive information about colleges in India. Whether you're a student looking for the best college to pursue your education or a parent seeking information about colleges for your child, this application aims to make the process easier by offering valuable insights into various educational institutions across the country.

## Features

- **College Information**: View detailed information about colleges, including their names, locations, accreditation status, and more.

- **Sort Functionality**: Sort colleges based on their CollegeDunia rating, fees, user review rating, and placement statistics.

- **Search Capability**: Easily search for colleges by name using the search functionality.

- **Compare Fees**: Compare the fees of different colleges side by side to make informed decisions.
